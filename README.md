# saucedemo
Test Suite de Cypress para https://www.saucedemo.com/

## Overview
Test automatizados para [www.saucedemo.com](https://www.saucedemo.com/) requerido, usando Cypress con JavaScript.

## Tecnologias Usadas:
- Cypress
- JavaScript
- Mochawesome Reporter

## Instalación y Ejecución
### 1. Clonar Repo:

`git clone https://github.com/JackDanielson/demoqa.git`
`cd saucedemo`

### 2. Instalar las Dependencias necesarias:

`npm install`

### 3. Correr Tests:

`npx cypress run` - Headless mode
`npx cypress open` - Interactive mode

## Reportes

Los Reportes serán generados en /cypress/reports folder después de la ejecución.
